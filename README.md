# Symbiote.js Isomorphic App Reference

> Here you can find code examples and reference for building isomorphic apps with Symbiote.js.

## How to run

```bash
npm install
npm run ssr
```

- then open `./dist/index.html`.

## What's inside?

Here we provide the set of examples of using Symbiote.js for building complex isomorphic / hybrid apps:

- Isomorphic web-components
- Computed properties
- Icons
- Routing
- Dynamic localization
- Dynamic section loading
- Application data management
- Todo-list example via Symbiote Itemize API (todo)
- Server-side only components (todo)
- Client-side only components (todo)
- SSR for Shadow DOM (DSD) components (todo)
- Dynamic theme switching (todo)
- Micro-frontend simplified example with code sharing via importmaps (todo)
- CSS data property initialization `--prop: 'value'` (todo)
- Shared state bindings (todo)
- Bubbling state bindings (todo)
- Named state bindings

## What is excluded for simplicity?

> Important! This example is simplified for demo and educational purposes.

- Build step (bundling and minification for html, js, css)
- Localization maps dynamic loading and optimization (index based l10n maps building)
- Asset generation for multiple endpoints (server-side routing)
